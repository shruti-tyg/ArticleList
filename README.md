# ArticleList Description
- Article List and detail for iOS.
- Written in Swift.

#### For CocoaPods User
Add pod 'IBAnimatable'
   pod 'SDWebImage'
   pod 'SVProgressHUD'
   pod 'Alamofire'
   pod 'SwiftyJSON'
   to Podfile.

# How To Use
#### Step 1
Run "Pod install" Comment in terminal 
#### Step 2
Run project workspace to test the code
